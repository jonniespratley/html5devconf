Hundreds of Generators

Just about a generator for everything, generators are installed via Node’s package manager NPM. No matter what project your working on there is most likely a generator for it.

A generator for generators

Use the yo generator to create a custom generator that runs in a Node.js environment. Leverage the yeoman-generator API to add advanced functionality to new or existing generators.
